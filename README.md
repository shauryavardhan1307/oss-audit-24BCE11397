## 📌 Project Overview
This project is an audit of the open-source software Mozilla Firefox.  
It covers the origin, license, ethics, Linux usage, ecosystem, and comparison with proprietary software.

---

## 💻 Shell Scripts

### 1. System Identity Report
Displays system information such as user, kernel version, uptime, and date.

### 2. FOSS Package Inspector
Checks whether Firefox is installed and displays its version and description.

### 3. Disk and Permission Auditor
Displays directory permissions and disk usage.

### 4. Log File Analyzer
Counts occurrences of a keyword (like "error") in a log file.

### 5. Open Source Manifesto Generator
Generates a personalized open-source statement based on user input.
